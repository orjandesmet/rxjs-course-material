# RxJS Course material

## How to use with npm

```bash
# install all dependencies.
npm i

# Serve the app:
npm start

# run unit tests:
npm test
```

## How to use with yarn

```bash
# install all dependencies.
yarn

# Serve the app:
yarn start

# run unit tests:
yarn test
```

## Exercises

The exercises revolve around a virtual car factory.
Check out branch `exercises` for an overview of all exercises.
